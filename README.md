## Features

- Easy to implement
- Easy to understand
- Shake detector
- Siren sound
- Call to registered mobile
- Shake device to send SOS to registered mobile and play siren
- Sends Last Known Location to registered mobile
- Now we can add multiple contacts to send SOS
